## 🔧 独立分离方法

如果未来需要将俄语专区分离为独立项目，只需要以下几步：

### 方法一：简单分离（推荐）
1. 将 `apps/ru/` 目录复制到新的代码仓库
2. 将 `shared/` 目录也复制到新仓库根目录
3. 修改所有资源引用路径，将 `../../shared/` 改为 `./shared/`
4. 修改首页链接，移除对主站的引用
5. 配置独立的部署流程

### 方法二：完整分离
1. 创建新的独立仓库
2. 复制以下目录和文件到新仓库：
   - `apps/ru/*` → 新仓库根目录
   - `shared/*` → 新仓库 `shared/` 目录
   - `vite.config.js` → 新仓库根目录（调整配置）
   - `package.json` → 新仓库根目录（调整配置）
3. 调整 Vite 配置，移除主站相关的页面入口
4. 配置独立的 GitHub Actions 部署流程
